# **Notify App**

### Backend – (Steps to start)

-   npm install
-   include .env file

    AWS_ACCESS_KEY_ID,

    AWS_SECRET_ACCESS_KEY,

    REGION,

    PORT,

    MAILGUN_KEY,

    MAILGUN_DOMAIN

-   npm start

### Frontend – (Steps to start)

-   npm start

MongoDB database should be locally installed and run on localhost:27017 which creates a db (notify).
